# Assignment ATM
(Coding) ให้นักศึกษาสร้างเว็บแอปพลิเคชันโดยใช้ Flask framework\

สร้างบัญชีผู้ใช้\
ดูยอดเงินในบัญชี\
ฝากเงินเข้าบัญชี\
ถอนเงินจากบัญชี\
ลบบัญชีผู้ใช้\

หมายเลขบัญชี (account_number) - string\

ยอดเงินในบัญชี (balance) - float\
